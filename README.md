# Blog Application

A full-stack Blog Application built using Spring Boot, Thymeleaf, PostgreSQL, and AWS services.

## Features

* Create, edit, and delete blog posts
* Store blog data using PostgreSQL
* Upload blog images to Amazon S3
* Responsive UI using Bootstrap
* RESTful backend with Spring Boot

## Technologies Used

* Java 17
* Spring Boot
* Spring Data JPA
* PostgreSQL
* Thymeleaf
* Bootstrap
* AWS S3
* AWS RDS
* AWS EC2
* Maven

## Architecture

User → Spring Boot Application → PostgreSQL Database (RDS)

Images → Amazon S3

## Future Improvements

* User Authentication & Authorization
* Comments System
* Likes & Reactions
* Docker Deployment
* CI/CD using GitHub Actions
